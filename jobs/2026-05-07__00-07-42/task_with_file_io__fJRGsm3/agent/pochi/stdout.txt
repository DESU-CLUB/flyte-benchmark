Task: `read_csv` Define an async task that reads a CSV file from the given path and returns a list of dicts: ```python @env.task async def read_csv(path: str) -> list: with open(path, newline="") as f: reader = csv.DictReader(f) return [dict(row) for row in reader] ``` ### 3. Task: `filter_high_scores` Define an async task that keeps only records where the `score` field (cast to `int`) is strictly greater than `threshold`: ```python @env.task async def filter_high_scores(records: list, threshold: int) -> list: return [r for r in records if int(r["score"]) > threshold] ``` ### 4. Task: `write_results` Define an async task that serialises the filtered records as JSON to `output_path` and returns the path: ```python @env.task async def write_results(records: list, output_path: str) -> str: with open(output_path, "w") as f: json.dump(records, f) return output_path ``` ### 5. Task: `csv_filter_pipeline` Define an async task that chains the three tasks above: ```python @env.task async def csv_filter_pipeline(input_path: str, output_path: str, threshold: int) -> str: records = await read_csv(input_path) filtered = await filter_high_scores(records, threshold) result_path = await write_results(filtered, output_path) return result_path ``` ### 6. `__main__` block ```python if __name__ == "__main__": asyncio.run(csv_filter_pipeline( "/home/user/flyte_project/input.csv", "/home/user/flyte_project/filtered.json", 80 )) ``` ## Implementation Guide 1.
