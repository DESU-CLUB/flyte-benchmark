---

### Architecture

```
input_text
    │
    ▼
split_into_shards()        ← pure Python helper; slices tokens into N equal buckets
    │
    ├─ shard 0 ─►  count_shard()  ─┐
    ├─ shard 1 ─►  count_shard()  ─┤  asyncio.gather (all shards run concurrently)
    ├─  ...                         │
    └─ shard N ─►  count_shard()  ─┘
                                    │
                                    ▼
                           reduce_counts()     ← merges Counter dicts, sorts by freq desc
                                    │
                                    ▼
                     /home/user/flyte_project/wordcount.json
```

---

### Flyte 2.0 API usage

| Construct | Purpose |
|---|---|
| `flyte.TaskEnvironment(name=…, resources=flyte.Resources(cpu=2, memory="512Mi"))` | Declares compute resources shared by all tasks in this environment |
| `@env.task` | Registers `count_shard`, `reduce_counts`, and `word_count_pipeline` as Flyte tasks |
| `asyncio.gather(*[count_shard(shard) for shard in shards])` | Map phase — all 8 shard coroutines are scheduled concurrently on the event loop |
| `await reduce_counts(shard_counts)` | Reduce phase — merges per-shard `Counter` dicts into the final sorted frequency table |
