### Key design decisions

| Concern | Solution |
|---|---|
| **Caching** | `cache=True, cache_version="v1.0"` on every task |
| **Type safety** | All dataclasses use Flyte-native types; models persisted as `FlyteFile` (pickle) |
| **Data transport** | `pd.DataFrame` → Parquet via PyArrow |
| **Parallelism** | Three training tasks are independent nodes — Flyte schedules them concurrently |
| **Selection metric** | Composite = 0.4×accuracy + 0.3×F1 + 0.3×ROC-AUC, with MCC > 0 guard |
| **Validation** | Hard errors raise, halting the pipeline; warnings are logged |
