Task: `generate_dataset` Define an async task: ```python @env.task async def generate_dataset(n_samples: int, n_features: int) -> dict: return { "X": [[float(i + j) for j in range(n_features)] for i in range(n_samples)], "y": [i % 2 for i in range(n_samples)], "n_samples": n_samples, "n_features": n_features, } ``` ### 3.
