The directory already exists and is writable. --- # Flyte 2.0 — ML Training Pipeline with Metrics ## Background Flyte 2.0 uses native Python async primitives for composing multi-step ML pipelines. Tasks are `async def` functions decorated with `@env.task` from a `TaskEnvironment`. Parallelism is achieved with `asyncio.gather`, and expensive steps can be memoised with `flyte.Cache`. ## Requirements Create the file `/home/user/flyte_project/ml_pipeline.py` that implements a complete ML training system which trains multiple model configurations in parallel and selects the best one. ### 1. Setup - Import `asyncio`, `datetime`, `json`, and `flyte`. - Create a `TaskEnvironment` named `"ml-training"` with `flyte.Resources(cpu=4, memory="8Gi")` (assign it to a variable named `env`): ```python env = flyte.TaskEnvironment("ml-training", resources=flyte.Resources(cpu=4, memory="8Gi")) ``` ### 2. Task: `generate_dataset` Define an async task: ```python @env.task async def generate_dataset(n_samples: int, n_features: int) -> dict: return { "X": [[float(i + j) for j in range(n_features)] for i in range(n_samples)], "y": [i % 2 for i in range(n_samples)], "n_samples": n_samples, "n_features": n_features, } ``` ### 3. Task: `train_model` (with cache) Define an async task with a `flyte.Cache` of 1 hour TTL: ```python @env.task(cache=flyte.Cache(ttl=datetime.timedelta(hours=1), version="v1")) async def train_model(config: dict, dataset: dict) -> dict: accuracy = min(0.99, config["learning_rate"] * config["epochs"] * 0.1) loss = max(0.01, 1.0 - config["learning_rate"] * config["epochs"] * 0.1) return { "config": config, "accuracy": accuracy, "loss": loss, "n_samples": dataset["n_samples"], } ``` `config` has keys: `model_type` (str), `learning_rate` (float), `epochs` (int). ### 4. Task: `select_best_model` Define an async task: ```python @env.task async def select_best_model(models: list) -> dict: best = max(models, key=lambda m: m["accuracy"]) return {"best_model": best, "n_candidates": len(models), "best_accuracy": best["accuracy"]} ``` ### 5. Task: `generate_report` Define an async task: ```python @env.task async def generate_report(best: dict) -> dict: return { "report_type": "model_selection", "winner": best["best_model"]["config"]["model_type"], "accuracy": best["best_accuracy"], "candidates_evaluated": best["n_candidates"], } ``` ### 6. Task: `run_ml_experiment` Define an async task that orchestrates the full pipeline: ```python @env.task async def run_ml_experiment(n_samples: int, n_features: int) -> dict: dataset = await generate_dataset(n_samples, n_features) configs = [ {"model_type": "lr", "learning_rate": 0.01, "epochs": 100}, {"model_type": "rf", "learning_rate": 0.1, "epochs": 50}, {"model_type": "nn", "learning_rate": 0.001, "epochs": 200}, ] trained = await asyncio.gather(*[train_model(cfg, dataset) for cfg in configs]) best = await select_best_model(list(trained)) report = await generate_report(best) return report ``` ### 7. `__main__` block ```python if __name__ == "__main__": report = asyncio.run(run_ml_experiment(1000, 10)) with open("/home/user/flyte_project/experiment_report.json", "w") as f: json.dump(report, f) ``` ## Implementation Guide 1.
